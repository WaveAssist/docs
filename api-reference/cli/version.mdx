# `waveassist version`

Display version information about the WaveAssist CLI and your environment.

## Usage

```bash
waveassist version
```

## Description

The `version` command displays information about your WaveAssist CLI installation and environment, including:
- CLI version
- Python version
- Operating system details

This information is useful for troubleshooting and when reporting issues.

## Options

This command has no additional options.
