# Node Operations

These endpoints allow you to manage nodes within your WaveAssist AI assistant projects, including creation, updates, and code management.

## Fetch Nodes

Retrieve all nodes for an AI assistant project.

```endpoint
GET /manage/fetch_nodes
```

### Query Parameters

```json
{
	"project_key": "string"
}
```

## Create Node

Create a new node in your AI assistant project.

```endpoint
POST /manage/create_node
```

### Request Body

```json
{
	"project_key": "string",
	"node_name": "string",
	"node_type": "string",
	"description": "string",
	"configuration": {
		"inputs": [],
		"outputs": [],
		"parameters": {}
	}
}
```

## Update Node

Update an existing node's configuration.

```endpoint
POST /manage/update_node
```

### Request Body

```json
{
	"project_key": "string",
	"node_key": "string",
	"node_name": "string",
	"description": "string",
	"configuration": {
		"inputs": [],
		"outputs": [],
		"parameters": {}
	}
}
```

## Delete Node

Remove a node from your AI assistant project.

```endpoint
POST /manage/delete_node
```

### Request Body

```json
{
	"project_key": "string",
	"node_key": "string"
}
```

## Update Code

Update the code associated with a node.

```endpoint
POST /manage/update_code
```

### Request Body

```json
{
	"project_key": "string",
	"node_key": "string",
	"code": "string",
	"language": "string"
}
```
