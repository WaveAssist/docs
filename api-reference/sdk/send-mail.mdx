Learn how to **send emails** using the WaveAssist SDK.

## Basic Usage

```python
import waveassist

# Initialize WaveAssist with your credentials
waveassist.init(
    "your-api-token",
    "your-project-key"
)

# Send a simple HTML email
waveassist.send_mail(
    to_email="recipient@example.com",
    subject="Test Email",
    html_content="<p>This is a test email sent from WaveAssist.</p>"
)
```

## Function Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `to_email` | `str` | Recipient's email address |
| `subject` | `str` | Subject line of the email |
| `html_content` | `str` | HTML content of the email body |

## Example with Custom Content

```python
import waveassist

# Initialize WaveAssist
waveassist.init(
    "your-api-token",
    "your-project-key"
)

# Send an email with formatted HTML content
waveassist.send_mail(
    to_email="bob_alice@gmail.com",
    subject="Terminal Test Email",
    html_content="<p>This is a test email sent from terminal.</p>"
)
```

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `to_email` | `str` | Yes | Recipient's email address |
| `subject` | `str` | Yes | Subject line of the email |
| `html_content` | `str` | Yes | HTML content of the email body |
