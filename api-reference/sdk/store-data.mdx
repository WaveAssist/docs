Learn how to **store different types of data** using the WaveAssist SDK for your AI assistants.

## Basic Usage

```python
import waveassist

# Store a simple string
waveassist.store_data("welcome_message", "Hello, World!")

# Store a dictionary
user_data = {"name": "Alice", "age": 30}
waveassist.store_data("user_profile", user_data)
```

## Storing DataFrames

```python
import pandas as pd
import waveassist

# Create a sample DataFrame
df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie"],
    "score": [95, 88, 92]
})

# Store the DataFrame
waveassist.store_data("user_scores", df)
```

## Function Parameters

| Parameter   | Type   | Required | Description                                     |
| ----------- | ------ | -------- | ----------------------------------------------- |
| key         | string | Yes      | Unique identifier for the data                  |
| value       | any    | Yes      | Data to store (string, number, dict, DataFrame) |
| environment | string | No       | Override default environment                    |
