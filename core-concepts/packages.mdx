---
title: "Packages"
description: "Learn about managing packages in WaveAssist."
---


**Packages** allow you to manage dependencies for your project. You can add, update, or remove packages as needed.

### Key Features
- **Add Packages**: Install new packages for your project.
- **Update Packages**: Keep your packages up-to-date.
- **Remove Packages**: Uninstall packages that are no longer needed.
