## Getting Started with WaveAssist

WaveAssist offers two powerful ways to work with your AI workflows: through our web dashboard and via local development using our Python SDK.

### Dashboard Development

<AccordionGroup>
  <Accordion icon="browser" title="Create Your First Project">
    1. Sign up at [WaveAssist Dashboard](https://app.waveassist.io)
    2. Click the "Add Project" card
    3. Fill in your project details
    4. Access your project key from the project settings

    <Frame caption="Create new project">
      <img
        className="rounded-lg shadow-lg dark:shadow-neutral-900"
        src="/images/create_project.png"
        alt="WaveAssist Project Creation"
      />
    </Frame>
  </Accordion>

  <Accordion icon="code" title="Edit Node Code">
    Each node in your workflow can be edited directly in the dashboard's code editor:

    <Frame caption="Dashboard Code Editor">
      <img
        className="rounded-lg shadow-lg dark:shadow-neutral-900 border border-neutral-200 dark:border-neutral-700"

    The editor provides:
    - Syntax highlighting
    - Code completion
    - Real-time error checking
    - Direct node execution
  </Accordion>

  <Accordion icon="gear" title="Node Configuration">
    Configure node settings directly in the dashboard:
    ```json
    {
      "name": "DataProcessor",
      "type": "processing",
      "schedule": "every 12 hours",
      "inputs": ["raw_data"],
      "outputs": ["processed_data"]
    }
    ```
  </Accordion>
</AccordionGroup>

### Local Development

<AccordionGroup>
  <Accordion icon="terminal" title="Install WaveAssist SDK">
    Install our Python package to start developing locally:
    ```bash
    pip install waveassist
    ```
  </Accordion>

  <Accordion icon="puzzle" title="Initialize WAHelper">
    Connect to your project using the WAHelper:
    ```python
    from waveassist import WAHelper

    # Initialize with your project key
    helper = WAHelper('your-project-key')
    ```
  </Accordion>

  <Accordion icon="code" title="Node Development">
    Create and manage nodes programmatically:
    ```python
    # Create a custom node
    @helper.node(type=NodeType.PROCESSOR)
    def process_data(input_data):
        # Your processing logic here
        return processed_data

    # Schedule node execution
    helper.schedule_node('process_data', interval='12h')
    ```
  </Accordion>
</AccordionGroup>

## Working with Data

```python
# Example workflow with data processing
from waveassist import WAHelper

# Initialize helper
helper = WAHelper('your-project-key')

# Configure data processing
helper.set_option('display.max_columns', None)

# Make API requests
url = 'https://api.example.com/data'
headers = {
    'Accept': 'application/json',
    'Connection': 'keep-alive'
}

# Process data
response = helper.process_data(url, headers)
```

## Best Practices

1. **Version Control**: Keep your node code in version control
2. **Testing**: Test nodes locally before deploying
3. **Security**: Never commit project keys to code
4. **Modularity**: Break complex workflows into smaller nodes

<Note>
Remember to keep your project key secure and never commit it to version control.
You can manage your keys in the project settings section of the dashboard.
</Note>
